ИИ который идиально подойдет для встройки в робота

Данная нейросеть написана на .NET (Quecto Roger 1 и ниже по архитектуре LP — .NET 8.0, Quecto Roger 2 и выше по архитектуре MLP — .NET 9.0)

Инструкция к Quecto Roger 2.0 / Yocto Roger 2.1 на Windows

1. Установить .NET 9.0

2. Запустить программу Quecto Roger v.2.0.exe / Yocto Roger v.2.1.exe

3. Отредактировать файл знаний.знать или создать свой файл знаний (подробнее в wiki)

4. Настройки: LR (Learning Rate) - чем меньше тем качественее будет обучаться нейросеть. Проходит - чем больше тем лучше усвоит информацию нейросеть. Файл знаний - файл знаний, на котором будет обучаться нейросеть. В Yocto Roger 2.1.exe есть параметр DropOut - это функциональная нейросеть от переобучения, советуется или ставится 5% или 10%, но с DropOut требуется более маленький LR и больше проходов.

Инструкция к Yocto Roger 2.1 на ARM aarch64 (Linux)

1. В папке с файлами ввести команду /.Yocto_roger_2.1

2. Отредактировать файл знаний knowledge.know (как их редактировать написано в wiki)

3. Настройки: LR (Learning Rate) - чем меньше тем качественее будет обучаться нейросеть. Проходит - чем больше тем лучше усвоит информацию нейросеть. Файл знаний - файл знаний, на котором будет обучаться нейросеть. В Yocto Roger 2.1.exe есть параметр DropOut - это функциональная нейросеть от переобучения, советуется или ставится 5% или 10%, но с DropOut требуется более маленький LR и больше проходов.
