# Roger

Моя нейросеть, которую я улучшаю каждый день.

Copyright (c) 2025 Axolotl512. Лицензия: MIT.

ИИ, который подходит для встройки в робота.

## Версии и архитектура

- **Quecto Roger 1 и ниже** — архитектура LP, .NET 8.0.
- **Quecto Roger 2 и выше** — архитектура MLP, .NET 9.0.
- **Yocto Roger 2.1** — архитектура MLP, .NET 9.0.

## Запуск на Windows (Quecto Roger 2.0 / Yocto Roger 2.1)

1. Установите .NET 9.0.
2. Запустите `Quecto Roger v.2.0.exe` или `Yocto Roger v.2.1.exe`.
3. Отредактируйте файл знаний или создайте собственный (подробнее в wiki).

### Параметры обучения

- **LR (Learning Rate)** — чем меньше значение, тем аккуратнее обучение.
- **Проходы** — чем больше значение, тем лучше сеть усваивает информацию.
- **Файл знаний** — файл, на котором обучается нейросеть.
- **DropOut** (только в `Yocto Roger v.2.1.exe`) — защита от переобучения.
  Рекомендуемые значения: **5%** или **10%**.
  При использовании DropOut обычно нужен меньший LR и больше проходов.

## Запуск на Linux ARM (aarch64, Yocto Roger 2.1)

1. В папке с файлами выполните:
   `./Yocto_roger_2.1`
2. Отредактируйте файл знаний `knowledge.know` (подробнее в wiki).

### Параметры обучения

- **LR (Learning Rate)** — чем меньше значение, тем аккуратнее обучение.
- **Проходы** — чем больше значение, тем лучше сеть усваивает информацию.
- **Файл знаний** — файл, на котором обучается нейросеть.

> ⚠️ Для ARM-версии Yocto Roger 2.1 / 3.0 установка .NET 9 не требуется.
